# Admin Dashboard 🔳

A modern, responsive administrative dashboard interface built as part of The Odin Project - https://www.theodinproject.com/lessons/node-path-intermediate-html-and-css-admin-dashboard Intermediate HTML and CSS curriculum. 

This project focuses on mastering **CSS Grid** and **Flexbox** to create a complex layout that remains organized and maintainable.

Live preview: https://llani-rainey.github.io/odin-admin-dashboard/

## ✨ Features

- **Complex Grid Layout:** Utilizing `grid-template-areas` and `grid-template-columns` for a sidebar-and-main-content structure.
- **Dynamic Project Cards:** A responsive grid of project cards that adjust based on screen size using `auto-fit`.
- **Sidebar Navigation:** A sticky sidebar featuring high-contrast links and a distinct branding area.
- **Interactive Elements:** Hover states for buttons and navigation items for a polished user experience.
- **Widgets:** Specialized sections for "Announcements" and "Trending" users using nested flexbox layouts.

## 🛠️ Built With

* **HTML5** - Semantic structure.
* **CSS3** - Custom properties, CSS Grid, and Flexbox.
* **Google Fonts** - 'Segoe UI' system stack for a clean, professional look.

## 💡 What I Learned

While building this dashboard, I deepened my understanding of:
- **Grid vs. Flexbox:** Knowing when to use Grid for the overall "skeleton" and Flexbox for the "organs" (like the header items and card icons).
- **Alignment:** Mastering `align-items` and `justify-content` to keep the user profile and search bar perfectly centered.
- **Z-Index & Shadows:** Using box-shadows and z-index to create a sense of depth between the header and the main content.
